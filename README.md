# 习思想题库刷题

一个纯前端题库刷题页面，支持多题库切换、随机组卷、考试模式、选项乱序、快捷键显示答案和逐题解析展示。

## 功能

- 题库选择：支持题库1、题库2、题库3、题库4。
- 练习模式：从当前题库随机抽取 15 道单选、25 道多选。
- 考试模式：从全部题库去重后的题池随机抽取 80 道单选、20 道多选。
- 答案查看：点击“显示答案”或按 `p` 显示/隐藏答案。
- 全部题库：支持查看、搜索、筛选全部题目。
- 解析展示：已整理的联网核验解析会自动匹配到对应题目。

## 本地运行

直接用静态文件服务器打开本目录即可：

```bash
python3 -m http.server 8765
```

然后访问：

```text
http://127.0.0.1:8765/
```

## 文件说明

- `index.html`：页面入口。
- `刷题网页.css`：页面样式。
- `刷题网页.js`：刷题逻辑。
- `题库列表.json`：题库清单。
- `题库.import.json`、`题库2.import.json`、`题库3.import.json`、`题库4.import.json`：题库数据。
- `联网核验解析.json`：已整理的解析数据。
